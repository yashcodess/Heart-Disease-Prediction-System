# Heart Disease Prediction System

A machine learning based web app that predicts heart disease risk using health parameters.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- SQLite
- HTML/CSS/JS

## Features
- Heart disease prediction
- Risk level detection
- Dashboard analytics
- CSV export
- Prediction history

## Run Project

pip install -r requirements.txt
python app.py
